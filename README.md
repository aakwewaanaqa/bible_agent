# 聖經小幫手 (Bible Agent)

一款使用 Flutter 開發的聖經閱讀與搜尋應用程式，專為教會弟兄姊妹設計。

## 功能特色

- 📖 快速搜尋聖經書卷
- 🔍 支援繁體中文書卷名稱
- 📱 跨平台支援（Android、iOS、網頁、桌面）

## 如何安裝

### 一般使用者

請至發布頁面下載適合您裝置的版本。

### 開發者

1. 確認已安裝 [Flutter](https://flutter.dev/docs/get-started/install)
2. 複製專案：
   ```bash
   git clone https://github.com/aakwewaanaqa/bible_agent.git
   cd bible_agent
   ```
3. 安裝相依套件：
   ```bash
   flutter pub get
   ```
4. 執行應用程式：
   ```bash
   flutter run
   ```
